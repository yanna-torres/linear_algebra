
# Linear Algebra

This repository contains Python implementations of several numerical methods for solving linear systems, matrix decompositions, and eigenvalue/eigenvector computation, developed as part of a practical study on Computational Linear Algebra.

## Project Structure

```bash
numerical_methods/
│
├── main.py                # Main script for running examples
├── methods/               # Implementations of algorithms
├── utils/                 # Helper functions
└── examples/              # Test cases and example matrices
```
